## chrome 插件（直接打开google搜索结果链接，不经过Google服务器）

**使用Google 搜索时 ，打开搜索链接，经常需要先跳转到Google中间服务器，再去到目标地址，增加了打开目标链接的时间，尤其是在大陆，翻墙极不稳定。**

### 所以就写了一个小插件绕开Google服务器，直接跳转到目标地址，加快目标地址打开时间。

### 只需30行
